Name

tc35892_work — bottom half handler for tc35892

Synopsis

void tc35892_work (struct work_struct *  _chip);

Arguments

_chip

pointer to the work_struct

Description

This is a work queue scheduled from the interrupt handler. This funtion reads from the ISR and finds out the source of interrupt. At the moment the handler handles only GPIO interrupts. Once it figures out the interrupt is from GPIO, a subsequent read is performed on isgpior register to find out the gpio number which triggered the interrupt. This funtion also clears the interrupt after handling.
