Open the error log view

The "Error Log" view can be opened with "Help > Show Error Log" or with the ALT+E shortcut. This view shows the exceptions caught by Trace Viewer workbench.

Please note that the "Caught exception: Connection reset" is normal, it occurs when you kill the decoder process ( button). Any other logged line is suspicious. In case you want to report an error, generate an error log archive with the "Generate Trace Viewer log..." tool (see the section below).



Generate Trace Viewer log

You can access this button in "Help > Generate Trace Viewer log...". Once pressed, a simple dialog is opened. Enter the path where you want to save your error log file (*.zip) and press OK. A message is displayed to indicate the result of the operation.
