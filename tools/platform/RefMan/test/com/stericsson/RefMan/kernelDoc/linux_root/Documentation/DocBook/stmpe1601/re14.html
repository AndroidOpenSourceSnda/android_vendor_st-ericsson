stmpe1601_gpio_set_value
Prev Chapter 3. Internal Functions Provided Next

Name

stmpe1601_gpio_set_value — Set a GPIO value from the given offset

Synopsis

void stmpe1601_gpio_set_value (struct gpio_chip *  gc,
 unsigned  off,
 int  val);

Arguments

gc

pointer to the gpio_chip strcture

off

The write gpio offset

val

data to be set, 0 or 1, represents high or low state of pin

Description

This funtion is called from the gpio library to set/unset a GPIO value. This funtion sets the GPSR (Set Register) to set a bit.
