Install instructions for FlashKit GUI RCP (standalone)
======================================================

1. Install Java 1.5 on your machine if not installed yet.
2. To run FlashKit GUI standalone application enter to "flashkit" folder and run flashkit

Note:
Please configure the application settings before the ME is attached!
The configuration should be done in few steps:
1. Open preference page by clicking on "Windows" menu item and then "Preferences" menu item.
2. Select "Flash Kit" item in the tree and select "flash tool backend" item.
For more information how to configure the application use F1 or click on the button with question mark
