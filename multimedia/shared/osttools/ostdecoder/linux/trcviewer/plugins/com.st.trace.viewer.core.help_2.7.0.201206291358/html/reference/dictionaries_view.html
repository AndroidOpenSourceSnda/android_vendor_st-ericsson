Trace Viewer Workbench

Documentation > References > Descriptors View


Description

This view displays the loaded descriptors once you started a trace session. This view is refreshed wit hthe new descriptors when a new trace session starts.



Menu

Icon Name Description
"Export as CSV" button Exports the content of this view as a CSV file. It opens a simple file dialog to choose the file to save the csv to.
"Revert to Default" button Reverts the settings of this view to default (column sorting, column width).


Tips

  1. Open source file:

    Expand the locations and paths items to get access to the file items. Double-click on the blue ones to open the corresponding source file. If the file is found an editor is openend and the file is annotated where trace packets have been emitted. You can also open source file from the instances items.

  2. Instance format:

    The format for instance is "<class>::<method>". Instances can be double-clicked to access to the corresponding source file at the correct line.
