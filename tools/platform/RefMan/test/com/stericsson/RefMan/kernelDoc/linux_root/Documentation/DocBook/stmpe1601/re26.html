Name

stmpe1601_read_info — read the chip information

Synopsis

int stmpe1601_read_info (void); 

Arguments

None

Description

This function read stmpe1601 chip and version ID and returns error if chip id or version id is not correct. This function can be called to check if UIB is connected or not.
