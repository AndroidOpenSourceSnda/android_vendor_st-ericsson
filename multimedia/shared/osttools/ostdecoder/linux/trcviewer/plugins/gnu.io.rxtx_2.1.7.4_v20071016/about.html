About This Content

This is an unmodified binary release of RXTX-2.1-7r2 as downloaded from http://www.rxtx.org, with the following exceptions:
  • RXTXCommDriver.java v1.16.2.57 was compiled from RXTX CVS and included in order to support disabling version print on startup (set System Property -Dgnu.io.rxtx.NoVersionOutput=true). This is RXTX bug 71.
  • CommPortIdentifier.java: Include the Fix for RXTX bug 83 due to which CommPortOwnership events are lost. The original fix from bugzilla is included in this version of the library.
  • Linux native library was built without lockfiles (contrary to the original distribution) in order to allow easier use. See the file BUILD_INFO.txt in gnu.io.rxtx.linux.x86 and gnu.io.rxtx.linux.x86_64 for details.
  • Solaris native library was linked statically in order to avoid dependency on libgcc_s (RXTX bug 73). See the file BUILD_INFO.txt in gnu.io.rxtx.solaris.sparc for details.
