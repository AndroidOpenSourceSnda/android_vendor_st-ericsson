Detailed Description

C/C++ component constructor interface.

The lifecycle.constructor interface must be implement by a component in order to provide a constructor that will be called by the component manager when the component will be instantiated.

Such code in your primitive ADL component must be used to provide constructor: 

 primitive your.component.name {
     ...
     provides lifecycle.constructor as constructor
     ...
 }
