Detailed Description

C/C++ component stopper interface.

The lifecycle.stopper interface must be implement by a component in order to provide a stop that will be called by the component manager when the component will exit from RUNNABLE state.

Such code in your primitive ADL component must be used to provide stopper: 

 primitive your.component.name {
     ...
     provides lifecycle.stopper as stopper
     ...
 }
