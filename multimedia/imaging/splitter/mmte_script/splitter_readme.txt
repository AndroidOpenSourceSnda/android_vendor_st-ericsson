To run standalone Splitter tests:
Once the board has booted, lauch the test from the MMTE command line as follows:
MMTE> do $(IMAGING_PATH)\splitter\mmte_script\splitter_unit_test_full_attach_io.mmte
MMTE> do $(IMAGING_PATH)\splitter\mmte_script\splitter_input_attach_io_output_fakesinks.mmte
MMTE> do $(IMAGING_PATH)\splitter\mmte_script\splitter_with_camera_and_display.ite
MMTE> do $(IMAGING_PATH)\splitter\mmte_script\ut_debug.mmte
