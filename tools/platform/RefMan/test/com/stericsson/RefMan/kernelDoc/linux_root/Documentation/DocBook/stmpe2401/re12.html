Name

stmpe2401_probe — initialize the stmpe2401

Synopsis

int stmpe2401_probe (struct i2c_client *  client,
 const struct i2c_device_id *  id);

Arguments

client

pointer to the i2c client structure

id

pointer to the i2c device id table

Description

This funtion is called during the kernel boot. This set up the stmpe2401 in a predictable state and attaches the stmpe to the gpio library structure and set up the i2c client information
