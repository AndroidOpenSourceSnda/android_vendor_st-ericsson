LICENSES USED BY THE BACKEND
============================

Following products have been used by the Flash-tool-backend

* Eclipse plug-in (OSGI - Equinox, SWT)
* LibUSB
* Apache POI

Both Eclipse plug-in are distributed under EPL which is available at
http://www.eclipse.org/legal/epl-v10.html. EPL is also included inside
jar files for concrete Eclipse plug-in. As the EPL states all the
content including source codes of these plug-in can be obtained at
http://www.eclipse.org.

http://sourceforge.net/apps/trac/libusb-win32/wiki. The original code of the 
libusb-win32 library has been changed and the change fulfills requirements 
given by LGPL. The updated version of Libusb-win32 is delivered including its 
modified source codes under the same condition as the original version.
