Howto process a file to flash

Processing a file to flash (flashing) is a common task where some updated platform software will be written to persistent storage on target. A file that can be processed is called a flash archive and is created as a result of the post-build process when building the platform software or part of the platform software.

Pre-requisites

Please make sure the preferences are correctly configured as described in Flash Kit Preferences. Also make sure that:

  • A connection profile used for flashing is set as active connection profile
  • Flash Tool Backend accepts equipment connections

Process file to filash

Connect a mobile equipment and wait until it is in state "idle". There are a number of ways to execute the process file task:

  • Right-click on the connected mobile equipment and select "Process File to Flash...". Select file to flash from th file dialog.
  • Click on the "Process File to Flash" toolbar icon. Select file to flash from the file dialog
  • Drag a flash archive file from the eclipse project navigator and drop it on a connected equipment
  • Drag a flash archive file from outside eclipse and drop it on a connected equipment
